# Forløb 4 - Vektorer i 2D

## Del 7 : Rette linjer og vektorer

-----------------------------------------------------------------------------

Linjens ligning kender vi fra grundforløbet og grundskolen:

$y = a \cdot x + b$


#### Linjens beskrevet med vektorer:

Vi kan beskrive linjen vha. vektorer. Men inden da skal vi kende til følgende vektorer:

**Retningsvektor**: for ovenstående linje kunne det være $\vec{v} = \begin{pmatrix} 1 \\\ a \end{pmatrix}$.

**Normalvektor**: for ovenstående linje kunne det være $\vec{n} = \begin{pmatrix} a \\\ -1 \end{pmatrix}$.

**Linjens hældning**: er $ v = arcTan(\frac{r_2}{r_1}) $

**Ortogonale linjer**: Med vektorer kan $ a \cdot c = -1 $ nu bevises nemt

$ \text{(.1.) } \vec{r_1} \cdot \vec{r_2} = 0 $

$ \text{(.2.) } \begin{pmatrix} 1 \\\ a \end{pmatrix} \cdot \begin{pmatrix} 1 \\\ c \end{pmatrix}= 1 \cdot 1 + a \cdot c = 0 $

$ \text{(.3.) } a \cdot c = -1 $
