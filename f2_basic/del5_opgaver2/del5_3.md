## Mængder

#### Vigtig viden:

$ x \in A $ , x tilhører A , dvs. x er tal fra mængden A

 K = { $ x | p(x)  $ }  , x er tal i mænden K, symbolet "|" betyder "for hvilket det gældet at...", dvs. alle x skal opfylde betingelsen p(x)

 K = { $ x \in R | x > 2 $ } , dette eks. angiver en mængde K der indeholder reelle tal størrer end 2  

$ K = A \cap B $ , fællesmænde

$ K = A \cup B $ , foreningsmængde

#### Opgaver:

A = {1,2,3,4}  og B = {4,5,6,7}

M1.) Hvad er $ C = A \cap B$  og $D =  A \cup B$

M2.) K = { $x \in A | x^2 \in B$ }

M3.) M  = { $x \in A | x^2 \in A \cap B $ }
